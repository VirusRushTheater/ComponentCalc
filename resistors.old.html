Component value calculator

One of the most annoying problems an electronic designer has to deal with when designing a circuit is to choose which values are the most appropiate for their needs. This tool was designed with that purpose. Be it a voltage divider, a RC charge circuit, a voltage regulator feedback arrangement, this calculator is aimed for all those purposes. Enter which components do you have, the function you want and restrictions, and let this calculator do the rest.

Don't worry, this program will remember your component values (as long as you don't clear your browser data)

Function

Write here the function you want to use your components in. Accepted parameters are 'R', 'C' and 'L' followed by a number. For example you can put R1/(R2+R1) = 0.5 to search components for a voltage divider.

When you press 'Calculate' it will throw the list of components which make the best matches and their error percentage. Limit by default is the best 100 matches. You can use restrictions to limit which components to use.

Results

These are the results of the calculation.

Restrictions

Restrict which components to use here. For example, if you want R1 to be between 1k and 5k, enter two restrictions: R1 >= 1k and R1 <= 5k. Restrictions are limited for now to single component values, sorry.



Toolbox

Put here the values of the components you have in your toolbox, separated by commas. Values such as '4.7', '5.1k', '27uF' or '3k3' are accepted.
